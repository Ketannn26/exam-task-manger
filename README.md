# TaskFlow — Task Management System

## Tech Stack

- Next.js 16 App Router + TypeScript
- Redux Toolkit (global task state)
- Context API (theme/UI preferences)
- Tailwind CSS + shadcn/ui

## Setup Instructions

1. Clone the repo
2. Run `npm install`
3. Copy `.env.local.example` to `.env.local`
4. Run `npm run dev`
5. Open browser DevTools → Application → Cookies → add `auth_token = test123`
